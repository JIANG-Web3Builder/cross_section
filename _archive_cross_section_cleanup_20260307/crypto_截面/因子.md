# 量化因子库：加密货币 1小时截面策略专用 (Crypto 1H Cross-Sectional Factors)

**适用场景**：加密货币全市场（Top 50 流动性），1小时级别数据 (OHLCV)。
**适用模型**：树模型 (LightGBM / XGBoost / CatBoost)。
**核心逻辑**：结合 Alpha 101 的非线性逻辑与机构常用的统计特征，通过截面标准化（Cross-Sectional Z-Score）捕捉相对强弱。

## 第一部分：Alpha 101 精选因子 (Top 15)
> **来源**：WorldQuant Alpha 101
> **筛选标准**：纯量价逻辑、在非有效市场（Crypto）依然有效、适合树模型捕捉非线性特征。

| ID | 因子名称 | 核心逻辑 / 伪代码描述 | 树模型中的作用 |
| :--- | :--- | :--- | :--- |
| **Alpha 001** | 量价背离反转 | `Rank(Ts_ArgMax(SignedPower((Ret<0?StdDev:Close), 2), 5))` <br> 捕捉回报率与波动率的极端背离。 | 捕捉恐慌抛售后的反弹点。 |
| **Alpha 006** | 量价相关性 | `-1 * Correlation(Open, Volume, 10)` <br> 开盘价与成交量正相关往往预示短期见顶。 | 识别诱多陷阱。 |
| **Alpha 009** | 短期均线趋势 | `Delta(Close, 5)` 的变种逻辑。 | 捕捉短期惯性。 |
| **Alpha 012** | 量价剧烈反转 | `Sign(Delta(Vol, 1)) * (-1 * Delta(Close, 1))` <br> 成交量暴增 + 价格暴跌 = 强买入信号。 | 识别恐慌盘（Panic Sell）。 |
| **Alpha 013** | 量加权协方差 | `Rank(Covariance(Close, Volume, 5))` | 确认资金推动的真实性。 |
| **Alpha 023** | 高波动均值回归 | 价格突破20周期高点后，若波动率过高则看空。 | 防止高位追涨。 |
| **Alpha 026** | 动量与量能排序 | `Rank(Vol) * Rank(High)` 的相关性负值。 | 识别量价配合失效的顶部。 |
| **Alpha 028** | 盘中均价偏离 | 盘中均价与收盘价的偏离度，结合成交量衰减。 | 捕捉日内（Intraday）反转。 |
| **Alpha 040** | 波动成本 | 高波动 + 放量 = 反转信号。 | 识别筹码松动的时刻。 |
| **Alpha 049** | 相对强弱位置 | 价格处于过去 N 天区间的相对位置。 | 纯粹的趋势确认。 |
| **Alpha 053** | 影线博弈 | `((Close-Low)-(High-Close)) / (Close-Low)` <br> 类似上影线与下影线的力度对抗。 | 识别多空力量转换。 |
| **Alpha 054** | 开盘跳空 | 捕捉开盘跳空（Gap）后的过度反应修复。 | 利用市场的非理性情绪。 |
| **Alpha 060** | 夹板突破 | 价格在 High-Low 区间的位置 + 时间衰减权重。 | 识别突破形态的强度。 |
| **Alpha 098** | VWAP 回归 | `Rank(Corr(VWAP, Vol))` 与 `Rank(Delta(Open))` 的差值。 | 机构算法单造成的定价偏差。 |
| **Alpha 101** | 实体力度 | `(Close - Open) / ((High - Low) + 0.001)` <br> K线实体占总波幅的比例。 | 信号强度过滤器，实体越大信号越强。 |

## 第二部分：机构实战统计因子 (Top 35)
> **来源**：顶级量化基金实战库
> **特点**：逻辑简单、鲁棒性强、不易过拟合。树模型非常依赖这些基础特征来构建决策树的分裂节点。

### A. 动量类 (Momentum) —— 追逐趋势
16. **ROC_6**: 6小时价格变化率（短线冲高）。
17. **ROC_24**: 24小时价格变化率（日内趋势）。
18. **Bias_24**: `Close / MA(24) - 1`，乖离率，衡量超买超卖。
19. **Slope_12**: 过去12小时价格线性回归斜率（趋势角度）。
20. **RSI_14**: 经典相对强弱指数。
21. **RSI_24**: 24小时周期的 RSI，更适合 Crypto 全天候交易。
22. **MACD_Hist**: MACD 柱状图，捕捉趋势加速/减速（二阶导数）。
23. **TRIX**: 三重平滑均线，过滤噪音后的纯粹动量。
24. **CCI**: 商品路径指标，专捉极端行情。
25. **APO**: 绝对价格震荡器 (`EMA_Fast - EMA_Slow`)。
26. **CMO**: 钱德动量摆动指标，对趋势反应更敏感。

### B. 波动率类 (Volatility) —— 捕捉变盘
27. **ATR_14**: 平均真实波幅（绝对值）。
28. **NATR_14**: `ATR / Close`，**归一化波动率**，解决了不同币价无法比较的问题。
29. **StdDev_24**: 过去24小时收益率标准差。
30. **UI (Ulcer Index)**: 溃疡指数，衡量下行风险和回撤深度。
31. **BB_Width**: 布林带宽度。极窄时（Squeeze）预示大行情即将爆发。
32. **Parkinson_Vol**: 利用 `High` 和 `Low` 计算的波动率，比只用 Close 更精准。

### C. 量价与流动性 (Volume & Liquidity) —— 资金痕迹
33. **VWAP_Dist**: `Close / VWAP - 1`。价格偏离成交量加权均价的程度。
34. **OBV**: 能量潮，累积成交量流向。
35. **MFI**: 资金流量指标（带成交量的 RSI）。
36. **Vol_Ratio**: `Volume / MA(Volume, 24)`。今日是否放量。
37. **PVT**: 量价趋势指标。
38. **Liquidity_Ratio**: `Abs(Return) / Volume`。衡量流动性（Amihud Illiquidity 变种），值越大说明流动性越差，冲击成本越高。
39. **Davids_Ratio**: 买卖力道比率（需利用 Close 相对于 High/Low 的位置估算）。

### D. 统计分布特征 (Statistical Moments) —— 树模型最爱
> **注意**：这部分因子树模型权重通常最高。

40. **Skewness_24**: 偏度。过去24小时收益分布是否歪斜？暴跌前通常有异常。
41. **Kurtosis_24**: 峰度。是否出现肥尾（极端行情）？
42. **Z_Score_Close**: `(Close - MA) / StdDev`。标准分。
43. **Ts_ArgMax_24**: 距离过去24小时**最高价**过去了多久（小时数）。
44. **Ts_ArgMin_24**: 距离过去24小时**最低价**过去了多久（小时数）。
45. **Correlation_PV**: 价格和成交量的滚动相关系数（12周期）。

### E. K线形态特征 (Pattern)
46. **Shadow_Upper**: `(High - Max(Open, Close)) / Close`。上影线长度（抛压）。
47. **Shadow_Lower**: `(Min(Open, Close) - Low) / Close`。下影线长度（支撑）。
48. **Body_Size**: `Abs(Open - Close) / Close`。K线实体力度。
49. **Gap**: `(Open_t - Close_t-1) / Close_t-1`。跳空缺口幅度。
50. **Log_Return**: 对数收益率（作为基础输入）。

## 第三部分：Python 实现代码 (Feature Engineering)

以下代码基于 `pandas` 和 `pandas_ta`，可直接用于处理您的加密数据。

```python
import pandas as pd
import pandas_ta as ta
import numpy as np

def calculate_crypto_factors(df):
    """
    输入 df 必须包含 MultiIndex (Time, Asset_ID) 或单一 Asset 的 DataFrame。
    包含列: open, high, low, close, volume
    """
    # ----------------------------------------------------
    # 1. 基础技术指标 (Ta-Lib / Pandas-TA)
    # ----------------------------------------------------
    # 动量
    df['RSI_14'] = ta.rsi(df['close'], length=14)
    df['RSI_24'] = ta.rsi(df['close'], length=24)
    df['ROC_6'] = ta.roc(df['close'], length=6)
    df['ROC_24'] = ta.roc(df['close'], length=24)
    df['Bias_24'] = df['close'] / ta.sma(df['close'], length=24) - 1
    df['Slope_12'] = ta.slope(df['close'], length=12)
    df['CCI'] = ta.cci(df['high'], df['low'], df['close'], length=14)
    
    # 波动
    df['ATR_14'] = ta.atr(df['high'], df['low'], df['close'], length=14)
    df['NATR_14'] = df['ATR_14'] / df['close']  # 归一化非常重要
    df['BB_Width'] = ta.bbands(df['close'], length=20)['BBP_20_2.0'] # 使用 Bandwidth 或 %B
    
    # 量价
    df['MFI_14'] = ta.mfi(df['high'], df['low'], df['close'], df['volume'], length=14)
    df['VWAP'] = ta.vwap(df['high'], df['low'], df['close'], df['volume'])
    df['VWAP_Dist'] = df['close'] / df['VWAP'] - 1
    df['Vol_Ratio'] = df['volume'] / ta.sma(df['volume'], length=24)
    
    # ----------------------------------------------------
    # 2. 统计特征 (Rolling Stats - 树模型强特征)
    # ----------------------------------------------------
    # 滚动偏度与峰度
    df['Skew_24'] = df['close'].rolling(24).skew()
    df['Kurt_24'] = df['close'].rolling(24).kurt()
    
    # 时间位置 (ArgMax/ArgMin) - 极强的树模型特征
    # 计算距离最高/最低点过去了多少个bar
    df['Ts_ArgMax_24'] = df['high'].rolling(24).apply(lambda x: 24 - x.argmax() - 1, raw=True)
    df['Ts_ArgMin_24'] = df['low'].rolling(24).apply(lambda x: 24 - x.argmin() - 1, raw=True)
    
    # 量价相关性
    df['Corr_PV_12'] = df['close'].rolling(12).corr(df['volume'])

    # ----------------------------------------------------
    # 3. 形态与 Alpha 101 简化版
    # ----------------------------------------------------
    # 影线与实体
    df['Shadow_Upper'] = (df['high'] - df[['open', 'close']].max(axis=1)) / df['close']
    df['Shadow_Lower'] = (df[['open', 'close']].min(axis=1) - df['low']) / df['close']
    df['Body_Size'] = (df['close'] - df['open']).abs() / df['close']
    
    # Alpha 101 #006 (反转)
    df['Alpha006'] = -1 * df['open'].rolling(10).corr(df['volume'])
    
    # Alpha 101 #101 (力度)
    df['Alpha101'] = (df['close'] - df['open']) / ((df['high'] - df['low']) + 1e-9)

    return df
